# איך היא יודעת שמשהו עובד?

פשוט דברו בטבעיות. הבינה המלאכותית קוראת את התגובה שלכם, קובעת אם הייתם מרוצים, ומדרגת את הזיכרון ב-Roampal.

לדוגמה:

- אם אתם אומרים "תודה, זה פתר את זה" ← הזיכרון שעזר מקבל ציון גבוה יותר.
- אם אתם מתקנים את הבינה המלאכותית ← אותו זיכרון מקבל ציון נמוך יותר.
- אם אתם ממשיכים הלאה ללא תגובה ← אין אות חזק לאף אחד מהכיוונים.

אין כפתורים ללחוץ עליהם, אין דירוגים לתת. היא לומדת מהשיחה עצמה.
