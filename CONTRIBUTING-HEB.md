## 🛠 דרישות קדם

לפני שתתחילו, ודאו שמותקנים אצלכם הרכיבים הבאים:

- **Go**: גרסה 1.22.1 ומעלה.
- **Python**: גרסה 3.11 ומעלה (עם תמיכה ב-`venv`).
- **Docker / Podman**: עבור שירותים מבוססי קונטיינרים (PostgreSQL, Redis, ממשק משתמש קדמי וכו').
- **NVIDIA GPU**: נדרש עבור האצת חומרה (מומלץ NVIDIA RTX 3090 ומעלה).
- **CUDA Toolkit**: מוגדר כראוי על המארח או בסביבת WSL2.

## 🚀 מתחילים

1.  **שיבוט המאגר**:

    ```bash
    git clone https://github.com/oznav2/DictaChat.git
    cd BricksLLM
    ```

2.  **אתחול הסביבה**:

    ```bash
    cp .env.heb .env
    # ערכו את קובץ ה-.env כדי להגדיר סיסמאות ומפתחות API (Tavily, Perplexity וכו')
    ```

3.  **הורדת מודלים**:
    המערכת דורשת קובץ מודל בפורמט GGUF. ניתן להוריד אותו ידנית או לתת למתקין לטפל בכך.

    ```bash
    mkdir -p ./models
    wget https://huggingface.co/VRDate/DictaLM-3.0-24B-Thinking-FP8-Q4_0-GGUF/resolve/main/dictalm-3.0-24b-thinking-fp8-q4_0.gguf -O ./models/dictalm-3.0-24b-thinking-fp8-q4_0.gguf
    ```

4.  **הרצת המערכת**:
    ```bash
    ./start.sh
    ```

## 🏗 מבנה הפרויקט

- `cmd/bricksllm`: נקודת הכניסה הראשית לשרת/שער המבוסס על Go.
- `internal/`: ליבת הלוגיקה, כולל מתאמים (adapters) לספקי LLM שונים, ניהול מטמון (caching) ואחסון.
- `frontend-huggingface/`: ממשק הצ'אט המבוסס על React.
- `datagov/`: סקריפטים ב-Python לאינטגרציה עם נתוני ממשל.
- `mcp-sse-proxy/`: גישור שרתי MCP לממשק הצ'אט.
- `scripts/`: סקריפטים לעזר עבור פריסה (deployment) ותחזוקה.

## 💻 תהליך הפיתוח

### פיתוח ב-Go

- השתמשו בפורמט Go הסטנדרטי: `go fmt ./...`
- הרצת בדיקות (Tests): `go test ./internal/...`
- בניית הקובץ הבינארי: `go build -o bricksllm ./cmd/bricksllm`

### פיתוח ב-Python

- סביבה וירטואלית (`.venv`) נוצרת באופן אוטומטי על ידי `start.sh`.
- שמרו על קובץ `requirements.txt` או התקינו תלותיות דרך `pip`.

### פיתוח ממשק משתמש (Frontend)

- ממוקם ב-`frontend-huggingface/`.
- דרישות תהליך עבודה סטנדרטיות של React/Node.js.

### הוספת כלים חדשים (MCP)

- הפרויקט משתמש ב-Model Context Protocol (MCP) כדי להרחיב יכולות.
- עיינו ב-`mcp-sse-proxy` כדי לראות כיצד הכלים משולבים.
